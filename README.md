# kTech Garage Management System

Internal garage management system for **kTech Automotive**.

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS (custom dark UI)

## How to Run (Local)
```bash
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python run.py
